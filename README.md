# 逃课模拟器

一个基于校园像素地图的轻策略小游戏。  
玩家需要在寝室、图书馆、湖边路口、教学楼之间蹲点、冲刺、赌点名，在不被抓到的前提下把“只混到刚刚好”这件事做到极致。

## 当前玩法

- 固定道路移动，不能随便抄近路
- 可在路口或图书馆蹲点，等时机再冲教室
- 去 7 号教学楼必须经过图书馆方向
- 游戏内 1 分钟 = 现实 3 秒
- 开课前准备时间为 20 秒现实时间
- 不同课程有不同风险
- 老师状态、室友靠谱度、校园风声会影响当天局势
- 支持高风险课程二次点名
- 支持特殊事件，例如班长探头、老师提前进楼、食堂排队、室友现场直播

## 运行方式

这是一个纯前端静态项目，直接打开 `index.html` 即可运行。  
如果你习惯本地服务器，也可以在项目目录下用任意静态服务器启动。

## 项目结构

- `index.html`：页面结构
- `style.css`：像素风 UI 与地图样式
- `game.js`：核心游戏逻辑
- `assets/`：像素人物与建筑遮挡素材
